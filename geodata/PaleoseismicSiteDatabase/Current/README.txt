This folder contains the current/live/active versions of the Paleoseismic Site Database.
The database currently consists of a GIS shapefile and an Excel spreadsheet.

The naming convention is the version with the date posted.
Minor updates will be called V1.1 etc, Major updates V2 etc.
Previous versions are moved to the Archive folder.

V1 is the template/test version shared with team members.
V1.1 has some minor updates to values and comments and adding a Comments regarding Dating on the Slip Rate sheet.
